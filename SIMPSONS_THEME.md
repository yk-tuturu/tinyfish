# 🍩 TinyFish Simpsons Cartoon Theme

## Welcome to Springfield's Food Guide! 🎨

The TinyFish website has been completely redesigned in authentic **The Simpsons cartoon style**! Get ready for a yellow-tastic food discovery experience inspired by the longest-running American animated series.

## 🎨 Design Changes Overview

### Color Palette
The design now features the iconic Simpsons colors:
- **Primary Yellow** (`#FFD700`): The signature Simpsons yellow
- **Simpsons Orange** (`#FF8C00`): Homer's shirt color
- **Simpsons Blue** (`#1E90FF`): Classic Simpsons house color  
- **Simpsons Pink** (`#FF69B4`): Marge's hair/dress color
- **Simpsons Green** (`#00DD00`): Krusty's hair, Bart's shirt color
- **Cream/Peach** (`#FFF8DC`, `#FFE4B5`): Simpsons skin tone background
- **Black** (`#1a1a1a`): Bold outlines (iconic cartoon style)

### Typography
- **Font**: Comic Neue, Comic Sans MS, Fredoka (playful, rounded, comic-style)
- **Weight**: Extra bold (700-900) for that aggressive cartoon look
- **Text Shadows**: Added for depth and that 90s cartoon print feel

### UI Elements Styling

#### 🏷️ Buttons
- **Bold 4-5px solid black borders** (cartoon comic book style)
- **Thick drop shadows** (6-8px offset for depth)
- **Rounded corners** (18-30px for that friendly cartoon feel)
- **Playful hover effects** with rotation and scale
- **Gradient backgrounds** using Simpsons colors

#### 🎴 Cards
- **5px solid black borders** (authentic comic book outline)
- **Rounded corners** (25px)
- **Yellow-to-orange gradients**
- **6px drop shadows** for that pop-out effect
- **Cartoon hover animations** (rotate, scale)

#### 📝 Search Bar
- **5px solid black border**
- **30px border-radius**
- **White input fields with black text**
- **8px shadow effect**

#### 🗺️ Map Container
- **5px solid black border**
- **25px border-radius**
- **Yellow-orange gradient background**
- **Large drop shadows**

#### 📦 Loading Container
- **5px dashed black border** (comic book style)
- **30px border-radius**
- **Fun spinning cart animations**

### Animations Added
New cartoonish animations for that extra Simpsons feel:
- **cartoonBounce**: Exaggerated bouncing with scale
- **cartoonShake**: Violent shaking effect
- **cartoonSpin**: Spinning with scale variation
- **cartoonWobble**: Wobbling rotation
- **cartoonPop**: Pop-in effect for reveals

### Background
- **Gradient background**: Cream to peach to tan (`#FFF8DC` → `#FFE4B5` → `#FFDEAD`)
- Inspired by the Simpsons' house interior color palette

## 🎨 Design Philosophy

This redesign embraces the **iconic aesthetic of The Simpsons**:
1. **Bold outlines**: Everything has thick black borders (comic book style)
2. **Bright, flat colors**: No gradients that are too smooth; we want that cartoon feel
3. **Playful animations**: Bounces, rotations, and wobbles
4. **Comic Sans influence**: Using Comic Neue for that authentic comic book touch
5. **Drop shadows everywhere**: To give depth and that cardboard cutout feel

## 📝 Notes
